# CNC코리아 이카운트 ERP 자동화 제안서 (초안)

## 1. 프로젝트 배경

CNC코리아는 이카운트 ERP를 사용하여 판매, 구매, 재고, 급여, 채권 관리를 수행하고 있습니다.
현재 대부분의 작업이 수동으로 이루어지며, 반복적인 데이터 입력과 문서 작성에 상당한 시간이 소요됩니다.

## 3. 기술 솔루션

### 기획1: Chrome 확장프로그램
- 이카운트 ERP 페이지에서 직접 사용
- 사이드패널 UI로 파일 업로드, 워크플로우 실행
- 이카운트 Open API 직접 호출
- SheetJS/PDF.js로 파일 파싱, Claude API로 AI OCR

### 기획2: MCP 연동 플랫폼
- 독립 대시보드 (Next.js + FastAPI)
- 실시간 KPI 모니터링
- 정기 보고서 자동 생성/메일 발송
- MCP 서버를 통한 AI 에이전트 연동

## 4. 이카운트 Open API 활용

이카운트는 공식 Open API를 제공하며, 추가 비용 없이 사용 가능합니다.

**인증 흐름:**
1. Zone 조회 → 서버 URL 확인
2. 로그인 → SESSION_ID 발급
3. API 호출 → SESSION_ID 헤더 포함

**지원 모듈:** 판매, 구매, 재고, 제품, 거래처 등

## 5. 예상 효과

- **월간 시간 절감**: 약 60~80시간
- **데이터 정확도 향상**: 수동 입력 오류 제거
- **실시간 현황 파악**: 대시보드 KPI 모니터링
- **자동 보고서**: 일간/주간/월간 리포트 자동 생성
