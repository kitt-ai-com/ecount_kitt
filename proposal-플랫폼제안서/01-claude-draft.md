# CNC코리아 AI 업무자동화 플랫폼 도입 제안서

**제출일:** 2026년 4월 28일
**제출처:** CNC코리아
**제안사:** 킷에이아이(KiTT AI)

---

## 1. 제안 개요

### 1.1 배경
CNC코리아는 이카운트 ERP를 기반으로 판매, 구매, 재고, 채권, 인사 업무를 운영하고 있으나, 다수의 반복 업무가 수작업으로 진행되어 월 60~80시간의 비효율이 발생하고 있습니다.

본 제안은 **단순 자동화를 넘어**, 이카운트 ERP 데이터를 중심으로 한 **AI 통합 플랫폼**을 구축하여:
- 즉시 효과를 내는 **크롬 확장 프로그램** (Phase 1)
- 전사 데이터를 통합하는 **AI 대시보드 플랫폼** (Phase 2)
- 사내 모든 정보를 학습한 **AI 챗봇** (Phase 3)
- **물류센터까지 확장** 가능한 아키텍처 (Phase 4)

를 단계적으로 도입하는 로드맵을 제안합니다.

### 1.2 기대 효과 요약

| 구분 | 현재 | 도입 후 | 절감 |
|------|------|---------|------|
| 반복 업무 시간 | 월 80시간 | 월 8시간 | **90%** |
| 데이터 입력 오류 | 월 15~20건 | 월 1~2건 | **90%** |
| 보고서 작성 | 일 1시간 | 자동 생성 | **100%** |
| 신규 직원 온보딩 | 2주 | 3일 (AI 가이드) | **79%** |
| 의사결정 속도 | 1~2일 | 실시간 | **즉시** |

## 2. Phase 1: 크롬 확장 프로그램 (즉시 도입)

### 2.1 개요
이카운트 ERP 화면에서 바로 사용할 수 있는 Chrome Extension으로, 기존 업무 흐름을 변경하지 않고 자동화를 적용합니다.

### 2.2 핵심 기능

#### 주문·물류 자동화
| 기능 | 설명 | 시간 절감 |
|------|------|-----------|
| 카톡 주문 수집·입력 | 카카오톡 주문 → AI 파싱 → 이카운트 판매입력 자동 변환 | 30분→2분 |
| 엑셀 판매입력 업로드 | 엑셀 파일 → 자동 판매입력 일괄 등록 | 45분→3분 |
| 물류센터 양식 변환 | 이카운트 데이터 → 물류센터별 포맷 변환 → 메일 발송 | 20분→1분 |

#### 견적·영업 자동화
| 기능 | 설명 | 시간 절감 |
|------|------|-----------|
| 거래처별 단가 추천 | 품목별/거래처별 과거 단가 패턴 학습 → AI 추천 | 신규 |
| 자동 견적서 생성 | 견적서 자동 제작 + 이메일 전송 | 20분→3분 |
| 미전환 견적 알림 | 7일/14일 무응답 견적 자동 follow-up 알림 | 신규 |
| 가견적 (박스 추천) | B2B 치수 입력 → 최적 박스 추천 + 견적 자동 생성 | 신규 |

#### 결제·세금·매입 처리
| 기능 | 설명 | 시간 절감 |
|------|------|-----------|
| 세금계산서 매칭 체크 | 네이버페이/이니시스 결제 ↔ 이카운트 일치 여부 자동 체크 | 40분→5분 |
| 인보이스 OCR | PDF 인보이스 → AI OCR → 이카운트 구매입력 자동 변환 | 15분→2분 |

#### 재무·인사
| 기능 | 설명 | 시간 절감 |
|------|------|-----------|
| 채권현황 자동 갱신 | 매일 자동 갱신 (기존 매주 목요일 수작업) | 1시간→자동 |
| 급여 원클릭 자동화 | 급여대장 → 이체서 + 명세서 + 메일 일괄 처리 | 2시간→5분 |
| 영수증 OCR | 법인카드 영수증 → AI 항목 분류 → 비용 리스트 | 신규 |

### 2.3 기술 구성
```
Chrome Extension (Manifest V3)
├── Side Panel UI — 워크플로우 선택/파일 업로드/미리보기
├── Service Worker — 이카운트 API 호출/워크플로우 엔진
├── Content Script — 이카운트 화면 자동 입력 (API 미지원 폴백)
└── AI Engine — Claude API (OCR/파싱/추천)
```

### 2.4 도입 효과
- **변경 없이 즉시 적용**: 기존 이카운트 ERP 사용 흐름 그대로 유지
- **학습 비용 최소**: 사이드 패널에서 파일 업로드 → 미리보기 → 실행 3단계
- **데이터 보안**: 모든 데이터는 이카운트 Open API를 통해 직접 전송, 중간 서버 없음

## 3. Phase 2: AI 통합 플랫폼 (이카운트 연동 대시보드)

### 3.1 개요
이카운트 ERP 데이터를 실시간 수집·분석하는 **독립 웹 플랫폼**입니다. Chrome 확장의 자동화 기능에 더해, 전사 데이터 시각화·분석·보고서 자동 생성 기능을 제공합니다.

### 3.2 핵심 기능

#### 실시간 대시보드
- **KPI 카드**: 오늘 매출, 주문 수, 미수금, 재고 경고
- **매출 추이 차트**: 일/주/월/연 매출 비교 (Recharts)
- **채널별 매출 비중**: 스마트스토어 / 자사몰 / 카톡 B2B
- **거래처 활동성**: 활성 / 주의 / 휴면 / 이탈 등급 표시

#### 거래처 통계·분석
| 기능 | 설명 |
|------|------|
| 주문 이력 대시보드 | 거래처별 주문 이력/추이 한눈에 파악 |
| 매출 순위 (파레토) | 상위 20% 거래처가 전체 매출의 몇 % 차지하는지 |
| 활동성 등급 | AI 기반 거래처 등급 분류 (활성→주의→휴면→이탈) |
| 이탈 예측 | 주문 패턴 이상 감지 → 선제 대응 알림 |
| 결제 패턴 | 거래처별 평균 결제일, 연체 경향 분석 |
| 신규 vs 기존 비율 | 신규 거래처 유입률 추이 모니터링 |

#### 매출·품목 분석
| 기능 | 설명 |
|------|------|
| 일/주/월/연 매출 비교 | 전기 대비 증감률 자동 산출 |
| 채널별 매출 비중 | 스마트스토어/자사몰/카톡B2B 채널 분석 |
| 품목별 매출 순위 | 기여도/마진 분리 분석 |
| 시간대별·요일별 패턴 | 주문 피크 시간 파악 → 인력 배치 최적화 |
| 매출 목표 달성률 | 월별 목표 대비 실적 자동 트래킹 |
| 시즌성 분석 | 계절/시즌별 매출 패턴 → 재고 사전 확보 |

#### 자동 보고서
| 보고서 | 주기 | 내용 |
|--------|------|------|
| 일간 매출 요약 | 매일 18시 | 매출/주문/입금/특이사항 |
| 주간 채권 현황 | 매주 목요일 | 미수금 aging, 연체 거래처, 독촉 대상 |
| 월간 손익 보고서 | 매월 1일 | 매출/매입/마진/전월 대비 |
| 거래처 랭킹 | 월간 | 매출 TOP20, 신규, 이탈 위험 |

→ 이메일 자동 발송 + PDF 다운로드 + 대시보드 표시

## 4. Phase 3: 사내 AI 챗봇 (지식 플랫폼)

### 4.1 개요
CNC코리아의 **모든 사내 정보**를 학습한 AI 챗봇을 도입합니다. 직원 누구나 자연어로 질문하면 즉시 답변을 받을 수 있는 **사내 지식 비서**입니다.

### 4.2 학습 데이터 범위

```
사내 AI 챗봇 지식베이스
│
├── 📊 ERP 데이터 (이카운트 실시간 연동)
│   ├── 재고 현황 — "A0001 박스 재고 몇 개야?"
│   ├── 거래처 정보 — "홍길동 상사 최근 주문 내역?"
│   ├── 매출/매입 — "이번 달 매출 얼마야?"
│   ├── 채권 현황 — "연체 중인 거래처 리스트?"
│   └── 제품 정보 — "A형 박스 단가 얼마야?"
│
├── 📋 업무 매뉴얼
│   ├── 이카운트 사용법 — "판매전표 수정하는 방법?"
│   ├── 물류 프로세스 — "CJ대한통운 발송 절차?"
│   ├── 결제 처리 — "세금계산서 발행 방법?"
│   ├── 인사/총무 — "연차 신청 어떻게 해?"
│   └── 신규 직원 온보딩 — "첫 주에 뭘 해야 해?"
│
├── 📄 계약서·문서
│   ├── 거래처 계약서 — "A사 계약 조건 확인"
│   ├── 단가 계약 — "B사 올해 단가 계약 내용?"
│   ├── 물류 계약 — "택배비 단가 얼마?"
│   └── 내부 규정 — "출장비 정산 기준?"
│
├── 🏢 업체 관리
│   ├── 거래처 DB — "A사 담당자 연락처?"
│   ├── 공급업체 — "원지 공급업체 리스트?"
│   ├── 물류업체 — "제주도 배송 가능 업체?"
│   └── 협력업체 — "인쇄 외주 업체 추천?"
│
└── 💡 업무 노하우 (축적형)
    ├── 자주 묻는 질문 — 반복 질의 자동 학습
    ├── 문제 해결 사례 — "지난번 배송 클레임 어떻게 처리했지?"
    └── 베스트 프랙티스 — 숙련 직원의 암묵지 문서화
```

### 4.3 핵심 시나리오

#### 시나리오 1: 영업 담당자
```
직원: "홍길동 상사 최근 3개월 주문 내역 보여줘"
AI:   홍길동 상사 (거래처코드: C001)
      - 2026/04: A형 박스 500개, B형 200개 (총 1,200만원)
      - 2026/03: A형 박스 300개 (총 600만원)
      - 2026/02: A형 박스 400개, C형 100개 (총 950만원)
      ※ 전분기 대비 주문량 25% 증가 추세입니다.
      ※ 미수금 230만원 (결제예정일: 4/30)

직원: "그 거래처 단가 계약서 확인해줘"
AI:   홍길동 상사 2026년 단가 계약:
      - A형 박스: 2,000원/개 (작년 대비 5% 인상)
      - B형 박스: 3,500원/개 (동결)
      - 계약기간: 2026.01.01 ~ 2026.12.31
      [계약서 원본 링크]
```

#### 시나리오 2: 물류 담당자
```
직원: "오늘 출고할 주문 리스트"
AI:   오늘 출고 예정 주문 8건:
      1. 홍길동 상사 — A형 500개 (CJ대한통운)
      2. 김영수 — B형 50개 (한진택배)
      ...
      ※ A형 박스 현재 재고: 2,300개 (출고 후 1,800개)
      ※ 재고 부족 예상일: 5/15 (현 소진 속도 기준)

직원: "A형 박스 재고 부족하면 자동 발주해줘"
AI:   자동 발주 규칙 설정:
      - 품목: A형 박스 (PROD-A001)
      - 최소 재고: 1,000개
      - 발주 수량: 3,000개
      - 공급업체: 원지산업
      설정하시겠습니까? [확인] [수정]
```

#### 시나리오 3: 경영진
```
대표: "이번 주 매출 요약"
AI:   금주 매출 요약 (4/21~4/27):
      - 총 매출: 4,850만원 (전주 대비 +12%)
      - 주문 건수: 47건 (전주 42건)
      - 채널: 스마트스토어 45% / 카톡B2B 35% / 자사몰 20%
      - 톱3 품목: A형(35%), B형(28%), C형(15%)
      ※ 스마트스토어 매출 급증 — 프로모션 효과 추정

대표: "올해 매출 목표 달성률은?"
AI:   2026년 매출 목표: 12억
      - 현재 누적: 4.2억 (달성률 35%)
      - 4월 예상: 1.1억 (목표 1.0억 대비 110%)
      - 연간 예측: 12.8억 (현 추세 유지 시 목표 초과)
      [상세 보고서 다운로드]
```

```
│  │  │업무매뉴얼│ │계약서 │ │업체정보│  │  │
│  │  └────────┘ └────────┘ └────────┘  │  │
│  └─────────────────────────────────────┘  │
│                                           │
│  ┌─────────────────────────────────────┐  │
│  │     실시간 데이터 연동               │  │
│  │  이카운트 API ← 재고/매출/채권/제품 │  │
│  └─────────────────────────────────────┘  │
└───────────────────────────────────────────┘
```

**핵심 기술:**
- **RAG (검색 증강 생성)**: 사내 문서를 벡터화 → 질문과 관련된 문서를 검색 → AI가 정확한 답변 생성
- **Tool Use**: AI가 이카운트 API를 직접 호출하여 실시간 데이터 조회
- **MCP 서버**: Claude Desktop/웹에서 이카운트 도구를 자연어로 사용

### 4.5 도입 효과
- **즉시 답변**: "재고 몇 개?" → 3초 내 응답 (기존: ERP 로그인 → 메뉴 탐색 → 조회 = 2~3분)
- **온보딩 단축**: 신규 직원이 AI에게 물어보며 업무 학습 (2주 → 3일)
- **암묵지 보존**: 숙련 직원의 노하우가 AI 지식으로 축적
- **24시간 운영**: 야간/주말에도 재고 확인, 주문 상태 조회 가능

## 5. Phase 4: 물류센터 확장

### 5.1 개요
CNC코리아 물류센터 운영을 AI 플랫폼에 통합하여, 주문→생산→출고→배송→정산까지 **End-to-End 자동화**를 구현합니다.

### 5.2 물류센터 통합 기능

#### 입고 관리
```
원자재 입고 → AI 자동 검수 → 이카운트 재고 반영
├── 바코드/QR 스캔 → 입고 수량 자동 입력
├── AI 이미지 검수 → 불량/파손 자동 감지
└── 공급업체별 입고 이력 → 납품 품질 분석
```

#### 재고·창고 관리
```
실시간 재고 모니터링 → AI 최적화
├── 로케이션 관리 — 창고 내 위치 추적
├── 안전재고 AI 예측 — 수요 패턴 학습 → 자동 발주점 조정
├── ABC 분석 — 품목별 회전율 기반 자동 등급화
├── 유통기한 관리 — 선입선출 자동 알림
└── 재고 실사 — 태블릿 바코드 스캔 → 차이 자동 보고
```

#### 출고·배송 관리
```
주문 접수 → AI 최적 피킹 → 자동 출고
├── 피킹 리스트 자동 생성 — 동선 최적화
├── 택배사 자동 연동 — CJ/한진/롯데/우체국 API
├── 송장번호 자동 입력 → 이카운트 + 카페24 동기화
├── 배송 추적 — 실시간 상태 업데이트
└── 배송 완료 → 자동 정산 → 채권 갱신
```

#### 물류 분석 대시보드
| 지표 | 설명 |
|------|------|
| 일일 입출고량 | 입고/출고 물량 실시간 모니터링 |
| 창고 가동률 | 공간 활용도, 적재율 분석 |
| 피킹 효율 | 건당 피킹 시간, 오피킹률 |
| 배송 리드타임 | 주문→배송완료 평균 소요시간 |
| 반품·클레임률 | 사유별 분류, 추이 분석 |
