제안 개요

단순 자동화를 넘어, AI 통합 플랫폼으로 진화하는 4단계 로드맵

CNC코리아는 이카운트 ERP를 기반으로 판매, 구매, 재고, 채권, 인사 업무를 운영하고 있으나, 다수의 반복 업무가 수작업으로 진행되어 월 60~80시간의 비효율이 발생하고 있습니다.

본 제안은 즉시 효과를 내는 크롬 확장 프로그램에서 시작하여, AI 대시보드 → AI 챗봇 → 물류센터 확장까지 단계적으로 도입하는 전략을 제시합니다.

Phase 1 — 크롬 확장 프로그램

기존 업무 흐름 변경 없이, 이카운트 ERP 화면에서 바로 자동화 적용 | 4주

즉시 도입 가능

이카운트 ERP를 열면 사이드 패널에서 파일 업로드 → 미리보기 → 실행 3단계로 자동화가 완료됩니다. 별도 시스템 학습이 필요 없습니다.

기능설명절감
카톡 주문 수집·입력카카오톡 주문 → AI 파싱 → 이카운트 판매입력 자동 변환30분→2분

Phase 2 — AI 통합 대시보드 플랫폼

이카운트 데이터를 실시간 수집·분석하는 독립 웹 플랫폼 | 8주

실시간 대시보드

KPI 카드(오늘 매출, 주문 수, 미수금, 재고 경고) + 매출 추이 차트 + 채널별 매출 비중 + 거래처 활동성 등급을 한 화면에서 확인합니다.

거래처 통계 · 분석

기능설명
주문 이력 대시보드거래처별 주문 이력/추이 한눈에 파악
매출 순위 (파레토)상위 20% 거래처의 매출 기여도 분석
활동성 등급AI 기반 거래처 분류 (활성→주의→휴면→이탈)
이탈 예측주문 패턴 이상 감지 → 선제 대응 알림
결제 패턴평균 결제일, 연체 경향 분석
신규 vs 기존 비율신규 거래처 유입률 추이 모니터링

매출 · 품목 분석

기능설명
기간별 매출 비교일/주/월/연 매출 전기 대비 증감률 자동 산출
채널별 매출 비중스마트스토어 / 자사몰 / 카톡B2B 채널 분석
품목별 순위기여도/마진 분리 분석
시간대별 패턴주문 피크 시간 파악 → 인력 배치 최적화
목표 달성률월별 목표 대비 실적 자동 트래킹
시즌성 분석계절별 매출 패턴 → 재고 사전 확보

자동 보고서

보고서주기내용

주간 채권 현황매주 목요일미수금 aging, 연체 거래처, 독촉 대상
월간 손익 보고서매월 1일매출/매입/마진/전월 대비

→ 이메일 자동 발송 + PDF 다운로드 + 대시보드 표시

Phase 3 — 사내 AI 챗봇

CNC코리아의 모든 사내 정보를 학습한 AI 지식 비서 | 8주

직원 누구나 자연어로 질문하면 즉시 답변을 받을 수 있는 사내 지식 플랫폼입니다.

학습 데이터 범위

ERP 데이터 (이카운트 실시간 연동)
재고 현황 — "A0001 박스 재고 몇 개야?"
거래처 정보 — "홍길동 상사 최근 주문 내역?"
매출/매입 — "이번 달 매출 얼마야?"
채권 현황 — "연체 중인 거래처 리스트?"

업무 매뉴얼
이카운트 사용법 — "판매전표 수정하는 방법?"
물류 프로세스 — "CJ대한통운 발송 절차?"
신규 직원 온보딩 — "첫 주에 뭘 해야 해?"

계약서 · 문서
거래처 계약서 — "A사 계약 조건 확인"
단가 계약 — "B사 올해 단가?"
내부 규정 — "출장비 정산 기준?"

업체 관리
거래처 DB — "A사 담당자 연락처?"
공급업체 — "원지 공급업체 리스트?"
물류업체 — "제주도 배송 가능 업체?"

업무 노하우 (축적형)
자주 묻는 질문 — 반복 질의 자동 학습
문제 해결 사례 — "지난번 클레임 어떻게 처리했지?"

활용 시나리오

시나리오 1: 영업 담당자
직원: "홍길동 상사 최근 3개월 주문 내역 보여줘"
AI: 홍길동 상사 (C001) - 4월: A형 500개, B형 200개 (1,200만원) - 3월: A형 300개 (600만원) - 2월: A형 400개, C형 100개 (950만원) ※ 전분기 대비 25% 증가 | 미수금 230만원 (결제예정 4/30)
시나리오 2: 물류 담당자
직원: "오늘 출고할 주문 리스트"
AI: 오늘 출고 예정 8건: 1. 홍길동 상사 — A형 500개 (CJ대한통운) 2. 김영수 — B형 50개 (한진택배) ※ A형 재고: 2,300개 (출고 후 1,800개) ※ 재고 부족 예상일: 5/15
시나리오 3: 경영진
대표: "이번 주 매출 요약"
AI: 금주 매출: 4,850만원 (전주 대비 +12%) 주문 47건 | 스마트스토어 45% / 카톡B2B 35% / 자사몰 20% 톱3: A형(35%), B형(28%), C형(15%) 연간 목표 달성률: 35% (현 추세 시 12.8억 예상)
