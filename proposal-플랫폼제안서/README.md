## 목적
CNC코리아 이카운트 ERP 자동화 + AI 플랫폼 도입 제안서 작성

## 협업 흐름
1. `01-claude-draft.md` — Claude 초안 (전체 제안서)
2. `02-codex-review.md` — Codex 리뷰 (보완점, 누락, 대안)
