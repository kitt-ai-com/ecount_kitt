- Login: `{zone}/OAPI/V2/OAPILogin`
- 판매: `{zone}/OAPI/V2/Sale/*`
- 구매: `{zone}/OAPI/V2/Purchase/*`

## 워크플로우 상세 (Notion 기획서 반영)

### 1. 주문 및 물류 관리
- **1-1. 카톡 주문 수집·입력** — 카톡 주문을 LLM 파싱 → 이카운트 판매입력 양식 자동 변환
- **1-2. 이카운트 자동 등록** — 엑셀 자동 판매입력 업로드
- **1-3. 물류센터별 포맷 자동 변환** — 이카운트 데이터 → 물류센터 양식 변환 → 메일 발송 (센터별 템플릿)

### 2. 견적·영업 자동화
- **2-1. 거래처별 단가 데이터화 및 추천** — 품목별/거래처별 단가 패턴 학습 → 추천
- **2-2. 자동 견적서** — 견적서 제작 + 이메일 전송
- **2-3. 미전환 견적 follow-up 알림** — 7일/14일 무응답 자동 알림
- **2-4. 가견적 기능** — B2B 치수 입력 → 최적 박스 추천 + 견적 자동 생성

### 3. 결제·세금계산서 처리
- **3-1. 세금계산서 매칭 체크** — 네이버페이/이니시스 결제 ↔ 이카운트 일치 여부 체크, 불일치 표시

### 4. 수입·매입 처리
- **4-1. Proforma Invoice OCR** — PDF → OCR+LLM 파싱 → 이카운트 구매입력 양식 변환

### 5. 재무·채권 관리
- **5-1. 채권현황 일일 자동 갱신** — 매일 자동 (기존 매주 목요일 수작업)
- **5-2. 월별 매입·매출 균형 리포트** — 추이 + 월별 손익 자동 산출

### 6. 인사·총무
- **6-1. 급여 업무 원클릭 자동화** — 급여대장 엑셀 → 기업은행 이체서 + 개인 명세서 + 메일
- **6-2. 비용 영수증 OCR** — 법인카드 영수증 사진 → 항목 분류 → 리스트

### 7. 거래처 통계·분석
- **7-1.** 거래처별 주문 이력 대시보드
- **7-2.** 거래처별 매출 순위 (파레토 분석)
- **7-3.** 거래처 활동성 등급 (활성/주의/휴면/이탈)
- **7-4.** 거래처 이탈 예측 (이상 패턴 감지)
- **7-5.** 거래처별 결제 패턴
- **7-6.** 신규 vs 기존 거래처 비율 추이

### 8. 매출·품목 통계
- **8-1.** 일/주/월/연 매출 비교
- **8-2.** 채널별 매출 비중 (스마트스토어/자사몰/카톡B2B)
- **8-3.** 품목별 매출 순위 (기여도/마진 분리 분석)
- **8-4.** 품목 카테고리별 매출
- **8-5.** 시간대별·요일별 주문 패턴
- **8-6.** 월별 매출 목표 대비 달성률
- **8-7.** 시즌성 분석
